## 表格相关配置如下
```
[
  {
    "label": "显示的列名",
    "prop": "字段名",
    "align": "单元格文本对齐方式",
    "headerAlign": "表头的文本对齐方式",
    "format": "一般用于时间的格式化,接收 moment 的格式化 format",
    "width": "列的固定宽度",
    "minWidth": "列的最小宽度",
    "isLink": "是否可点击",
    "filter": "过滤规则(数组,label,value), 适用于状态码转中文文字",
    "sortable": "开启表格排序(custom)",
  }
]
```
